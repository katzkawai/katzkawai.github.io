# katzkawai.github.io

## ポートフォリオサイト

ようこそ、わたしのポートフォリオサイトへ！このリポジトリはGitHub Pagesを使用して公開されているウェブサイトのソースコードを含んでいます。

## サイトの内容

- **メインページ**: プロフィール情報、最近の活動、プロジェクト紹介
- **Aboutページ**: サイトの目的と概要、使用技術の説明、更新履歴
- **各種プロジェクト**:
  - 静的サイトジェネレーター (Hugo, Jekyll, MkDocs, Lektor)
  - PWAサンプル
  - Reactサンプル
  - 愛犬のページ

## 使用技術

- HTML5
- CSS3
- Bootstrap 4
- JavaScript
- GitHub Pages
