KKLab — Katz Kawai の個人サイト

サイトの目的

KKLab は、Web 制作の練習と実験を記録するための個人サイトです。 新しい技術を試したり、作ったものの置き場所にしたりしています。

作者について

Katz Kawai。名古屋を拠点に、Web 開発を趣味として楽しんでいます。 軽くて速く、きちんと動くサイト作りが好きです。

使用している技術

  • 素の HTML5 / CSS3 / JavaScript によるモダンな実装
  • GitHub Pages でのホスティングと自動デプロイ
  • PWA（Progressive Web App）— オフライン対応・ホーム画面への追加
  • レスポンシブデザインとアクセシビリティへの配慮
