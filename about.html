本サイトについて

Katz Kawaiのポートフォリオサイト

サイト概要

サンプルサイトを作成しています。随時更新中。

練習でデータを追加してみました。

サイト編集ツール

GitHubページの編集には、Proseというオンラインエディターがおすすめです。

目的

このサイトは、以下の技術や手法を実験・紹介することを目的としています：

  • Bootstrapを用いたレスポンシブデザイン
  • GitHub Pagesでのホスティング
  • PWA（Progressive Web Apps）
  • Reactを用いたWeb開発

更新履歴

このサイトの主要な更新内容を時系列で記載しています。各更新は機能追加、コンテンツ追加、デザイン改善に分類されています。
